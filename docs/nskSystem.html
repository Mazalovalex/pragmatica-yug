Таблица цветов NCS – Natural Colour System

NCS — цветовая модель, разработанная Шведским Институтом Цвета. Основана на системе противоположных цветов. Широко используется в промышленности для описания цветов продукции. Состоит из цветов. В NCS цвет описывается шестью элементарными цветами: четыре хроматических (Желтый, Красный, Синий, Зеленый) и два нехроматических (Белый, Черный). Эти цвета нельзя описать сочетанием других.

3 простых шага для определения любого цвета в системе NCS

  • цвет

    На цветовом круге найдем цветовой тон: -R20B

  • оттенок

    Определим насыщенность и черноту. Используем Треугольник NCS для анализа.

  • Обозначение

    Сопоставив оттенок и цвет мы получаем итоготовое обозначение цвета в системе NCS
